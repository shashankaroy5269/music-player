A modern music player built with React.js, TypeScript, and Redux Toolkit.

Users can:

 1.Upload audio files

 2.Play / Pause music

3. Skip to the next track

4. Enjoy automatic playback of the next song

 * Experience smooth state management using Redux Toolkit

* Installation & Setup

Follow these steps to run the project locally:

1. Clone the Repository
git clone https://github.com/shashankaroy5269/music-player.git

2. Navigate to the Project Folder
cd music-player

3. Install Dependencies
npm install

4. Start the Development Server (Vite)
npm run dev

* Run the Project

After running the development server, open your browser and go to:

http://localhost:5173


Click the link in your terminal (Vite will show it), and the project will run in your browser.
